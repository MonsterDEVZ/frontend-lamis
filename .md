### Промпт для создания компонента "Наша история"

**Задача:** Создать для Next.js-приложения интерактивный React-компонент "Наша история" в виде горизонтального слайдера (таймлайна), который прокручивается по оси X при вертикальном скролле колесиком мыши (по оси Y).

**Технологии:**

- **Фреймворк:** Next.js
- **Язык:** TypeScript
- **Стилизация:** Tailwind CSS
- **Иконки:** Lucide React

---

### **1. Архитектура компонента**

Компонент должен состоять из двух основных частей:

1.  **Главный контейнер (`HistoryTimeline.tsx`):** Родительский компонент, который будет содержать всю логику скролла и отображать карточки.
2.  **Карточка таймлайна (`TimelineCard.tsx`):** Дочерний компонент для отображения информации о конкретном событии.

**Структура данных:**
Создайте массив объектов для хранения данных таймлайна. Первый элемент будет уникальным (вводная карточка), остальные — стандартными.

### **2. Логика горизонтального скролла**

Создайте кастомный хук `useHorizontalScroll.ts`, который будет преобразовывать вертикальный скролл в горизонтальный.

- Хук должен возвращать `ref` для DOM-элемента контейнера.
- Используйте `useEffect` для добавления и удаления обработчика события `wheel`.
- В обработчике события, если `deltaY` не равна нулю, предотвратите стандартное поведение (`e.preventDefault()`) и измените `el.scrollLeft += e.deltaY`.

```typescript
// /hooks/useHorizontalScroll.ts
import { useRef, useEffect } from 'react';

export const useHorizontalScroll = () => {
  const elRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    const el = elRef.current;
    if (el) {
      const onWheel = (e: WheelEvent) => {
        if (e.deltaY === 0) return;
        e.preventDefault();
        el.scrollTo({
          left: el.scrollLeft + e.deltaY,
          behavior: 'smooth', // Для плавной прокрутки
        });
      };
      el.addEventListener('wheel', onWheel);
      return () => el.removeEventListener('wheel', onWheel);
    }
  }, []);

  return elRef;
};
```

### **3. Реализация компонентов**

**`HistoryTimeline.tsx` (Главный контейнер):**

- Импортируйте `useHorizontalScroll` и примените возвращаемый `ref` к главному `div`-контейнеру.
- Используя `flex` и `overflow-x-auto` от Tailwind CSS, создайте горизонтально прокручиваемую область. Скроллбар нужно скрыть.
- С помощью `map` отрисуйте карточки из массива `historyData`.
- Стилизуйте контейнер так, чтобы он не занимал всю ширину экрана, и чтобы была видна часть следующей карточки для намека на возможность скролла.

**Примерная верстка:**

```jsx
import { historyData } from '@/data/historyData';
import { useHorizontalScroll } from '@/hooks/useHorizontalScroll';
import { TimelineCard } from './TimelineCard';
import { IntroCard } from './IntroCard'; // Отдельный компонент для первой карточки

export const HistoryTimeline = () => {
  const scrollRef = useHorizontalScroll();

  return (
    <div
      ref={scrollRef}
      className="flex overflow-x-auto cursor-grab space-x-8 py-10 pl-10 pr-20 no-scrollbar" // `no-scrollbar` - кастомный класс для скрытия скроллбара
    >
      {historyData.map((item, index) =>
        item.type === 'intro' ? (
          <IntroCard key={index} title={item.title} description={item.description[0]} />
        ) : (
          <TimelineCard
            key={index}
            year={item.year!}
            title={item.title}
            description={item.description}
            image={item.image!}
          />
        )
      )}
    </div>
  );
};
```

**`IntroCard.tsx` и `TimelineCard.tsx` (Карточки):**

- **Общие стили:** Карточки должны иметь фиксированную ширину и `flex-shrink-0`, чтобы они не сжимались.
- **`IntroCard`:**
  - Крупный заголовок "Наша история".
  - Параграф с описанием.
  - Иконка (например, `MouseWheel` из `lucide-react`) с текстом "Начните вращать колесико".
- **`TimelineCard`:**
  - Сверху изображение (`<Image>` из Next.js) с закругленными углами.
  - Под ним блок с годом (`2004`, `2006`...).
  - Текстовое описание события.
  - Стилизуйте "маркер" таймлайна (кружок с точкой внутри) слева от года.

**Пример `TimelineCard.tsx`:**

```jsx
import Image from 'next/image';

// ... props

export const TimelineCard = ({ year, title, description, image }) => {
  return (
    <div className="flex-shrink-0 w-[400px]">
      {' '}
      {/* Фиксированная ширина */}
      <Image src={image} alt={title} width={400} height={250} className="rounded-xl mb-4" />
      <div className="flex items-center space-x-4">
        {/* Маркер таймлайна */}
        <div className="flex-shrink-0 w-5 h-5 rounded-full border border-green-500 flex items-center justify-center">
          <div className="w-2 h-2 rounded-full bg-green-500"></div>
        </div>
        <h3 className="text-2xl font-bold">{year}</h3>
      </div>
      <div className="pl-9 mt-2">
        <p className="font-semibold">{title}</p>
        {description.map((line, i) => (
          <p key={i} className="text-gray-600 text-sm">
            {line}
          </p>
        ))}
      </div>
    </div>
  );
};
```

### **4. Стилизация (Tailwind CSS)**

- **Скрытие скроллбара:** Добавьте в `globals.css` кастомный класс для скрытия скроллбаров.
  ```css
  @layer utilities {
    .no-scrollbar::-webkit-scrollbar {
      display: none;
    }
    .no-scrollbar {
      -ms-overflow-style: none; /* IE and Edge */
      scrollbar-width: none; /* Firefox */
    }
  }
  ```
- **Типографика и цвета:** Используйте стили, соответствующие дизайну на картинке: черный жирный шрифт для заголовков, темно-серый для основного текста.
- **Отступы:** Настройте отступы (`padding`, `margin`, `space-x-*`) так, чтобы контент не прилипал к краям, и между карточками было достаточно "воздуха".
